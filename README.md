# 埃森梅莉亚 (Essenmelia)

欢迎来到 埃森梅莉亚 (Essenmelia) —— 一个记录与追踪你追过的故事、番剧、小说、电影、电视剧甚至事件的工具。

## 功能
- 导入和导出本地记录，轻松管理你的追踪清单
- 支持多种内容类型：事件、小说、番剧、电视剧、电影等
- 不作恶、跨平台

## 安装与使用（ChatGPT）
- 打开项目目录：cd D:\untitled\1
- 安装依赖：npm install
- 开发调试：npm run dev → 浏览器访问 http://localhost:3000
- 生产构建（可选）：npm run build:renderer
- 生产预览（可选）：serve dist
